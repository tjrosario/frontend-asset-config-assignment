# AI Rules

This project stores agent rules in topic-specific directories:

- `.codex/rules/general.md`
- `.codex/rules/testing.md`
- `.codex/rules/react.md`
- `.claude/rules/general.md`
- `.claude/rules/testing.md`
- `.claude/rules/react.md`
- `.cursor/rules/general.mdc`
- `.cursor/rules/testing.mdc`
- `.cursor/rules/react.mdc`

Read the relevant files before changing code. The assignment stack is Vite, React, strict TypeScript, React Hook Form, Zod, Material UI, Vitest, and React Testing Library.
